# person-reid-2019-NAIC
2019全国人工智能大赛提交代码
项目所需的工具包/框架
Python >= 3.5
PyTorch >= 0.4.0
TorchVision
Matplotlib
Argparse
Sklearn
Pillow
Numpy
Scipy
Tqdm

项目运行的资源环境
2卡11G-1080ti



项目的文件结构
Data：进行数据准备预处理
Model：建立模型
Loss:损失函数
Utils：工具函数

项目的运行步骤
1.运行train.py
2.运行main.py
3.在final_battle.json文件夹中含有生成的结果
